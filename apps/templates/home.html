CodeLn 3D models API

Fonctionnalités

API ROOT

Ce projet présente l'API model 3. Les fonctionnalités disponibles sont les suivantes:

  • lock Création de comptes utilisateur
  • lock Connexion et obtention de token jwt (Le token est passé en Header de toutes les route protegé)
  • lock Création de model3D: Seul les utilisateur connecté peuvent créer un model3D
  • lock Modification et suppression d'un models: Seul l'autheur peut modifier ou supprimer
  • lock Afficher la liste ou le détail d'un model
  • lock Afficher la liste ou le detail d'un utilisateur afin de voir ces badges

Background tasks

L'application contient deux type de background tacks

  • Hook (signal): cette tâche s'execute à la création d'un models 3D afin d'attribuer le badge collector à l'utilisateur si celui-ci remplir les conditions
  • Peridic tack (Celery): Ici nous avons deux tâche celery qui lance des commandes django à des interval de temps afin d'attribuer les bagdes Pionner et Start
